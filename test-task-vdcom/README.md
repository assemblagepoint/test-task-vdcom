#### Запуск:
```mvn package```

- #### Задание 1:
  * ```java -jar task1.jar```
- #### Задание 2:
  * ```java -jar task2.jar```
- #### Задание 3:
  * ```java -jar task3.jar``` - запуск с входными данными из примера в задании. 
  * ```java -jar task3.jar filename.txt``` - запуск с пользователькими входными данными, где ```filename.txt``` - название файла с входными данными, он должен находиться в каталоге модуля task3.

  Результат сохраняется в файле ```result.txt``` в каталоге модуля
- #### Веб-приложение:
  * ```java -jar webapp.jar``` и открыть в браузере localhost:8888
